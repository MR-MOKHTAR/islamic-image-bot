# 📸 HtmlSnapBot

ربات تلگرام برای تبدیل سریع و امن فایل‌های HTML به تصاویر PNG با کیفیت بالا. ایده‌آل برای تبدیل طرح‌های ساخته شده با هوش مصنوعی به پوستر و تصویر.

---

## ✨ قابلیت‌ها

- **تبدیل مستقیم HTML به تصویر**: فایل `.html` بفرستید و خروجی `.png` بگیرید.
- **امنیت بالا**: مسدود کردن اسکریپت‌های مخرب و درخواست‌های شبکه‌ای ناامن.
- **پشتیبانی از فونت‌های فارسی**: بارگذاری خودکار فونت‌ها از Google Fonts و cdnjs.
- **هوشمند و سریع**: رندرینگ بهینه با Puppeteer و مدیریت خودکار ابعاد تصویر.
- **پایداری**: سیستم دانلود فایل با قابلیت تلاش مجدد (Retry) در صورت بروز خطا.

---

## 🚀 راه‌اندازی

### ۱. پیش‌نیازها

- **Node.js** نسخه ۱۸ یا بالاتر
- **npm** یا **yarn**

### ۲. نصب وابستگی‌ها

```bash
cd islamic-image-bot
npm install
```

> ⚠️ با اولین اجرا یا نصب، Puppeteer مرورگر Chrome را دانلود می‌کند که ممکن است چند دقیقه طول بکشد.

### ۳. تنظیم محیط

۱. فایل نمونه را کپی کنید:

```bash
cp .env.example .env
```

۲. فایل `.env` را باز کرده و توکن ربات خود را از [@BotFather](https://t.me/BotFather) وارد کنید.

### ۴. اجرا

**حالت توسعه:**

```bash
npm run dev
```

**محیط عملیاتی (Production):**

```bash
npm run build
npm start
```

---

## 🤖 نحوه استفاده

۱. فایل `.html` خود را در چت تلگرام برای ربات ارسال کنید.
۲. ربات فایل را پردازش کرده و تصویر با کیفیت آن را برای شما می‌فرستد.

**نکات مهم:**

- حداکثر حجم فایل مجاز: **۲ مگابایت**
- اسکریپت‌های خارجی (JS) به دلایل امنیتی مسدود می‌شوند.
- تصاویر باید به صورت **Base64** در فایل HTML جاسازی شده باشند.

## 🛠 عیب‌یابی

**خطای Puppeteer در لینوکس:**
اگر با خطای نبود کتابخانه‌های سیستم روبرو شدید، دستور زیر را اجرا کنید:

```bash
sudo apt-get install -y \
  libgbm-dev libxkbcommon-x11-0 libgtk-3-0 \
  libnss3 libatk-bridge2.0-0 libdrm2 libxcomposite1
```

**فونت‌ها بارگذاری نمی‌شوند:**
ربات از Google Fonts و cdnjs استفاده می‌کند. مطمئن شوید سرور شما به این دامنه‌ها دسترسی دارد.
